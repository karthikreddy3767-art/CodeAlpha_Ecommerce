# CodeAlpha E-Commerce Website

## Features

- User Registration
- User Login (JWT Authentication)
- MongoDB Atlas Database
- Product Management
- Shopping Cart
- Checkout System
- Order History
- Admin Dashboard
- Product Search
- Responsive UI

## Technologies Used

- HTML
- CSS
- Bootstrap 5
- JavaScript
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

## Installation

```bash
npm install
npm run dev
```

Server:

```
http://localhost:5000
```
